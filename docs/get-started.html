Shopping List:

  • ESP32 Dev Kit
  • Solid State Relay

    One per heating zone, 40 amp or higher
  • Relay Heatsinks

    One per Solid State Relay
  • Contactor

    Acts as the main power switch for firing
  • Mechanical 5V Relay

    Turns the Contactor on and off
  • Ethernet Cable

    If you plan on mounting your PyKiln close to the kiln buy a short cable.
  • Adafruit Universal Thermocouple Amplifier MAX31856

    Only purchase this is you need more than one temperature sensor
  • High Temperature K-Type Thermocouple

    One per temperature sensor
  • Red and Black Wire

    25 feet of each should be plenty
  • PyKiln Control Board

    This isn't currently available for sale.
  • Used Electric Kiln

    Check both Facebook Marketplace and Craigslist.

All of the products above have Amazon affiliate links, which if purchased a small percentage goes back to supporting this project. I would advise you to also check Ebay, AliExpress, or other online stores as you might find a better price elsewhere.

Buy The PyKiln Kit:

  • PyKiln Kit

    This includes all of the items above except for the kiln, some parts may be slightly different due to availability. PyKiln comes pre-installed on the ESP32 and all of the sensors and relays have been tested and verified to be working. This isn't currently available for sale.
